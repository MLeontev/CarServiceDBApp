# Информационная система учета работ автосервиса

## Описание
Программа предоставляет интерфейс для управления содержимым базы данных автосервиса. Позволяет выполнять операции чтения, добавления, редактирования и удаления данных о клиентах, автомобилях, заказах, сотрудниках, услугах, а также формировать отчеты.

## Возможности
- **Учет клиентов**: ведение базы клиентов с указанием их автомобилей, хранение истории заказов.
- **Учет автомобилей**: хранение данных об автомобилях и истории их ремонтов.
- **Создание заказов**: оформление и отслеживание заказов с указанием клиентов, автомобилей, исполнителей, перечня работ, стоимости и сроков выполнения.
- **Учет выполненных работ**: ведение информации о выполненных работах.
- **Справочник услуг**: ведение справочника услуг с описанием и стоимостью.
- **Учет сотрудников**: хранение информации о сотрудниках, распределение работ.
- **Формирование отчетов**: автоматизированные отчеты по клиентам, заказам, мастерам и выполненным услугам.
- **Настройка прав доступа**:
  - **Директор**: управление персоналом, доступ ко всей информации в системе.
  - **Мастер-приемщик**: управление клиентами, заказами, распределение работ, создание отчетов.
  - **Автослесарь**: доступ к назначенным работам, отметка выполненных задач.

## Стек технологий
- **Язык программирования**: C#
- **СУБД**: MySQL
- **Доступ к данным**: ADO.NET
- **Интерфейс**: WinForms
- **Формирование отчетов**: EPPlus
